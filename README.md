# Модуль vt.forms для 1С-Битрикс

Универсальное решение для создания и обработки AJAX-форм с гибкой структурой данных.

## Основные возможности

- Встроенный компонент и контроллер: Готовая инфраструктура для отправки и обработки данных «из коробки».

- Гибкое хранение (EAV): Данные распределены по двум связанным таблицам (Результат и Значения). Это позволяет добавлять любое количество полей без изменения структуры базы данных.

- Современный ORM: Использование DataManager, связей OneToMany и Entity Collections (D7) для высокой производительности и чистоты кода.

- Административный интерфейс: Удобный просмотр результатов через стандартный компонент main.ui.grid.

## Установка

Клонируйте репозиторий в директорию модулей:

```bash
cd local/modules/
git clone https://github.com/n0firt/vt.forms.git
```

Установите модуль через административную панель:

`Рабочий стол` `→` `Marketplace` `→` `Установленные решения`

## Использование

### Регистрация формы

Формы регистрируются в конфигурации модуля через сервис `Repository` или в любом другом месте с использованием `ServiceLocator`:

```php
$repository->add(
    new Form(
        'request',
        new TextField(
            new FieldDto('NAME', 'Имя', true)
        ),
        new PhoneField(
            new FieldDto('PHONE', 'Телефон', true)
        ),
        new EmailField(
            new FieldDto('EMAIL', 'Email')
        ),
        new TextField(
            new FieldDto('MESSAGE', 'Комментарий', isTextArea: true)
        )
    )
);
```

### Вызов компонента

Компонент автоматически обеспечивает валидацию полей на основе их типов, заданных при регистрации.

```php
$APPLICATION->IncludeComponent(
    'vt:form',
    '',
    [
        'FORM_ID'     => 'request',
        'TITLE'       => 'Заявка',
        'TOP_TEXT'    => 'Оставьте заявку, и менеджер перезвонит вам в течение 5 минут',
        'BUTTON_TEXT' => 'Отправить заявку'
    ]
);
```

### Административный раздел

После установки будет доступен административный раздел

`Сервисы` `->` `AJAX формы` `->` `Результаты`
